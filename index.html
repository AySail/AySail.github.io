My Setup

At the moment, I’m coding and creating on a MacBook Air (13-inch, 2017) running macOS Monterey 12.7.6. Some of the main apps and tools I use include VS Code for coding, TurboWarp for Scratch-based projects, Figma and Inkscape for design and vector graphics, Zen Browser for browsing, Notion for organizing my ideas, Prism Launcher for managing Minecraft mods, plus the built-in macOS Calculator and Terminal for everyday tasks.
